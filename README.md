# Getting Started

## 1. Cloner le repo
```console
git clone https://github.com/FlamySpeeddraw/touchedelherbe
cd touchedelherbe
```

## 2. Installer les dépendances
```console
npm i
```

## 3. Lancer le projet
```console
npx expo start
```

Installer l'application EXPO GO sur google play/app store
